Situs Kidiko kembali tersedia dengan struktur Astro terbaru.

Rilis ini memulihkan halaman utama, halaman paket, blog, kontak, dan sitemap dari URL yang sudah terindeks.

### Features Included:

*   **Halaman terindeks:** Beranda, tentang, impact, toko, penawaran, TOEFL ITP, dan paket AI.
*   **Content Collections:** Configured for Blog and Changelog with MDX support.
*   **Styling:** Fully integrated with Tailwind CSS v4.
*   **Components:** Reusable UI elements like Buttons and Badges.
*   **Responsive Design:** Looks great on mobile, tablet, and desktop.
